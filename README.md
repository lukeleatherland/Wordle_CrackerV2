# Wordle_CrackerV2
Uses Information theory and information enthalpy to find the best wordle word and beat the wordle
To install run pip install requirements.txt
At the bottom of Wordle_crackerV2 you can change the keybindings for each function
By default the commands are:
1-capture top left
2-capture top right
3-A window will pop up showing the first row make sure it lines up
3-make guess
4-colour calibration(only need to use if the code is mixing up the yellow and green tiles)
5-exit
HOW TO USE
-hover your mouse over the top left of the wordle grid and push 1
-hover your mouse over the bottom right of the wordle grid and push 2
-push 3 to make a guess
-the code will end once the word is found
-Make sure you are in the window when click to guess as if not this cause the code to try and type and fail
