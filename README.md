# Wordle_CrackerV2\n
Uses Information theory and information enthalpy to find the best wordle word and beat the wordle\n
To install run pip install requirements.txt\n
At the bottom of Wordle_crackerV2 you can change the keybindings for each function\n
By default the commands are:\n
1-capture top left\n
2-capture top right\n
3-A window will pop up showing the first row make sure it lines up\n
3-make guess\n
4-colour calibration(only need to use if the code is mixing up the yellow and green tiles)\n
5-exit\n
HOW TO USE
-hover your mouse over the top left of the wordle grid and push 1\n
-hover your mouse over the bottom right of the wordle grid and push 2\n
-push 3 to make a guess\n
-the code will end once the word is found\n
-Make sure you are in the window when click to guess as if not this cause the code to try and type and fail
