To install, run:
pip install -r requirements.txt


Keybindings
At the bottom of Wordle_crackerV2 you can change the keybindings for each function.
Default commands:
- Capture top-left
- Capture top-right
- A window will pop up showing the first row — make sure it lines up
- Make guess
- Colour calibration (only needed if the code mixes up yellow and green tiles)
- Exit
How to Use
- Hover your mouse over the top-left of the Wordle grid and press 1
- Hover your mouse over the bottom-right of the Wordle grid and press 2
- Press 3 to make a guess
- The code will stop once the word is found
- Make sure the Wordle window is active when the program types the guess — otherwise it may fail
